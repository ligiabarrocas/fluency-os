FluencyOS

Plataforma de inglês prático para uma pessoa PM em tecnologia: fluência, pronúncia, conversação, escrita, gramática objetiva, rituais de projeto e vocabulário real de times de TI, Dados e IA.

🎯

Objetivo

Destravar fala, escuta, leitura e escrita para conduzir reuniões, escrever status e conversar com Dev, QA, Dados, Produto, Arquitetura e stakeholders.

🧭

Plano por nível

A2 para sobrevivência, B1 para conversas, B2 para decisões, C1 para liderança e C2 para nuance.

🎧

Pronúncia + áudio

Frases importantes têm inglês, português, pronúncia aproximada e áudio lento/normal.

Login e compartilhamento

No GitHub Pages, esta versão é pública e gratuita. O progresso fica salvo no navegador de cada pessoa. Login Google, cadastro real, área administrativa e termos aceitos por usuário exigem uma fase futura com backend seguro, como Firebase ou Supabase.
